# TYPING TRAINER
    This is a simple web app aimed at boosting the reaction timing for a key that appeared on the
    screen while being helpful for beginners to build a basic understanding of the keyboard.

### Project idea
    While I have been using computer devices for quite a few years, I used to type while gawking 
    down at my keyboard until just recently. Now I got an idea that instead of those standard 
    keyboard training apps why aren't there any other apps which help first get the layout of the
    keyboard down, instead of the apps which isolate the whole keyboard keys and then the user 
    slowly learns the keyboard bit by bit. I didn't know the reason but I knew that there could
    be an easy web implementation of my idea. 


## Features :
    - random character generation
    - 3 or more levels targeting different character sets
    - statistics of the typing session
    - real-time correct/input tracking
    - simple UI

## Technologies used: 
    - HTML
    - CSS
    - Javascript

## Working:
    This web app used random character generation for the level selected and displays it
    on the screen, on the basis of the input provided it displays real time correctness 
    of the input and shows statistics at the end of the session.
